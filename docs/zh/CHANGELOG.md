# 更新日志

Claude-Statusline-Pro-Max 的所有重要变更。

## [1.0.0] - 2025-04-30

### 新增

- 12 级响应式布局（L0-L11）+ 回退 + 紧急 + 最后手段
- 4 区域布局：模型、上下文、工作区、时长
- 基于 `od` 的精确 CJK/emoji 显示宽度计算
- 第三方 Provider 支持（优雅处理缺失的 `rate_limits`）
- 5 秒 Git 分支缓存，使用 `stat -f %m` 时间戳
- 复合时长格式（`1h24m`、`2d3h`）
- 上下文进度条三级颜色阈值（绿/黄/红）
- Vim 模式指示器（`[N]`/`[I]`/`[V]`/`[V-L]`）
- 思考指示器、努力等级、代理名称标记
- `try_build` 函数支持 `show_rate`/`show_vim`/`show_dur` 标志
- 路径为空时分支加 `│` 前缀（视觉区分）
- macOS bash 3.2 兼容（不使用 bash 4+ 特性）
- 所有终端宽度零溢出保证（542 测试用例）
